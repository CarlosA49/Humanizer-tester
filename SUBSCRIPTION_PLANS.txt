COUPONS & OVERRIDES
--------------------------------------------------------------------------------
   Coupon types ..... Percentage-off (% off)
                      Peso-off (PHP off)
                      100%-free
   Code signing ..... Codes signed with COUPON_SECRET
   Owner generator .. Password-gated (footer -> "Owner tools").
                      Owner password stored only as a SHA-256 hash
                      (plaintext never stored).
   Dev override ..... A separate SHA-256-gated developer "no limit"
                      password turns off the word limit on that ONE
                      device only (hidden feature).


IMPORTANT CAVEATS
--------------------------------------------------------------------------------
   - Device limits are a SOFT limit shown to users for now. Real
     per-device binding and tamper-resistant trial / coupon / word
     enforcement only apply once the Supabase backend phase is enabled.
     Without it, this is a bypassable client-side MVP.
   - Payment capture is MANUAL: PayMongo (GCash + cards -> BPI) or
     PayPal -> manual activation by the owner.
