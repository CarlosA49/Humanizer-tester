PayPal QR (international payments)
=================================

Drop your personal PayPal QR image here, named exactly:

    paypal-qr.png

The checkout screen shows it automatically for international customers.
Until the file exists, the app shows a graceful "QR will be added by the
owner" note instead (no error).

Optional: set PAYPAL_ME in docs/config.js to also show a PayPal.me link.
